# Nono

![](https://i.imgur.com/2N4NLnF.png)

## Setup

Setup your local configuration
```
cp config/private.exs.sample config/private.exs
```

Pull dependencies
```
mix deps.get
```

Run the local server
```
mix run --no-halt
```

## Contribute

```
mix test
```
